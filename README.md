# go-zts

#简介：

基于go语言实现的简单定时任务调度系统，有简易操作的用户界面，便于交互，支持大部分语言编写的任务执行，利用golang的goroutin控制子进程状态，实现任务调度。

#支持任务语言：
##Python
##Ruby
##Golang
##C++
##C
##Java
##PHP
##lisp
##bash


#安装使用：

##依赖：
go get github.com/alphazero/Go-Redis

go get github.com/coocood/jas
